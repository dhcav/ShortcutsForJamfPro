Created by Damian Cavanagh in August 2025. 

This Shortcut uses the ipsw.me API to allow easy downloading of Apple platform firmware.

You can search for firmware:
- By platform and version
- By device
- For the current device

If running on Mac, you can also move IPSW files from your Downloads folder to the Conifgurator cache for use in restoring devices. 

Third party app requirement:
a-Shell is required to get firmware for current device when run on mobile devices, as it is used to retrieve the model identifier from the MobileGestalt plist. The Shortcut will check if the app is installed and opens its App Store listing to quickly download. FWIW, it is annoying to have to use a third party app to do something that until late 2024 was possible to do natively with the Get Contents of URL Action.
